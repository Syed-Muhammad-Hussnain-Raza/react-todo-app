# React ToDo App

## Description

A simple ToDo application built using React.js. This app allows users to add and delete tasks. It demonstrates basic React functionality and state management.

## Features

- Add new tasks
- Delete tasks
- Responsive design

## Installation

### Prerequisites

- Node.js and npm

### Steps to Install

1. **Clone the Repository**:

   ```sh
   git clone https://github.com/Syed-Muhammad-Hussnain-Raza/react-todo-app.git
   ```

2. **Navigate to the Project Directory:**:

   ```sh
   cd react-todo-app
   ```

3. **Install Dependencies:**:
   ```sh
   npm install
   ```

### Usage

1. **Start the Project:**:

   ```sh
   npm run dev
   ```
